# Pokemon API Practice

A simple Python script to practice using APIs with the PokeAPI.

## Description
This project fetches Pokemon data from the PokeAPI and displays information like name, ID, height, and weight.

## Requirements
- Python 3.x
- requests library

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python APIPractice.py
```

## Example Output
```
Name: gengar
Id: 94
Height: 15
Weight: 405
```

## What I Learned
- How to make GET requests using the requests library
- Working with JSON data from APIs
- Basic error handling with status codes
